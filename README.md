# GrabiDown
#### Python script to temporarily "down" grabify links.

# Supported Domains
```
_common_domains = [
    "grabify.link",
    "photovault.pics",
    "bathtub.pics",
    "foot.wiki",
    "thisdomainislong.lol",
    "gamergirl.pro",
    "picshost.pics",
]
```

# Requirements
```py
pip install requests
```
```py
pip install user_agent
```
```py
pip install colorama
```
